RoverTrak

Project description

My capstone project for the Seattle University Web Development Certificate program was an educational web application called 'Rovertrak.' Rovertrak pulls data from the NASA Open API and allows users to learn about the three Martian rovers (Curiosity, Opportunity, and Spirit) and view the most recent images from these vehicles.

The project required me to use third-party API data in the context of an AngularJS framework. Although I completed the project for the purposes of the certificate program, I am continuing to develop it. In the next phase, I will use Ionic to port Rovertrak into a native mobile application. I will also be adding camera selection and other features

Technologies used
  • Angular JS
  • Angular UI-Bootstrap
  • Ionic mobile framework
  • HTML5
  • CSS3
  • NASA Open API

QuickBGG

Project description

QuickBGG is an AngularJS single-page web application that I built as the final project for the advanced Javascript course in the Seattle University Web Development Certificate program. I am a heavy user of BoardGameGeek.com, a forum-based website for board game enthusiasts, but always felt that the site was a little convoluted. I wanted to build a clean web app that allowed users to take advantage of BGG's more prominent features.

This first version of QuickBGG uses a third-party API to display BGG's 'Hotness' list, and also allows users to display and save BGG user game collections. Future features will include game trading and messaging features.

Technologies used
  • AngularJS
  • HTML5
  • CSS3
  • Bootstrap
